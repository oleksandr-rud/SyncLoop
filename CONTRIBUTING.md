# Contributing to SyncLoop

Thank you for your interest in contributing!

## Development

```bash
npm install
npm run typecheck
npm test
```

## Contribution Workflow

1. **Fork** the repository and create a new branch from `main`.
2. **Make your changes** — keep them focused and minimal.
3. **Run checks** — ensure `npm run typecheck` and `npm test` pass before submitting.
4. **Open a Pull Request** — describe what you changed and why.

Please follow the existing code style and architecture conventions described in `.agent-loop/`.
